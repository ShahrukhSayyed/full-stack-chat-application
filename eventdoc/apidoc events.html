Events to listen

verifyUser event
This event ("verifyUser") has to be listened on the user's end to verify user authentication.
user will only be set as online user after verification of authentication token.
online-user-list event
This event ("online-user-list") has to be listened on the user's end to identify all available users that are currently online.
All available users can be shown to the user based on which he can start chatting with a user.
The output will be an object, object has key as userId and value as full name.
userId event
This event's name is actually dynamic and refers to the userId of the logged in user.
This event ("userId") has to be listened to identify an individual chat message that has been received.
Example output it as follows:
                            {
                                chatId: 'unique chat id',
                                message: 'you message',
                                createdOn: date, 
                                receiverId: 'userId of receiver',
                                receiverName: full name of receiver,
                                senderId: 'userId of sender',
                                senderName: 'full name of sender'
                            }
error-occurred event
This event ("error-occurred") has to be listened to identify an error which will occur when the data is not sent properly in the events to be emitted.
Example output it as follows:
                            {
                                status: 123,
                                error: 'error message'
                            }
disconnect event (optional)
This event ("disconnect") has to be listened, when the user is disconnected to show proper message.
Auth Error (optional)
This event ("auth-error") has to be listened, when the user does not have valid auth token to do some tasks. So that it will show proper message to user.
created-chatroom Event
This event's is actually designed for listening event message after somone has created the Chat room (Group).
This event ("created-chatroom") has to be listened to identify whetehr the group is created by other user end. After listening to this event update the groups joined list and groups available list as well .
Example output it as follows:
                            {
                                chatRoomId:'unique chat room id',
                                chatRoomTitle: 'Name of chat room',
                                chatRoomLink: 'link of chat room',
                                userId: 'userId of creator',
                                userName: 'User who have created this chat room', 
                            }
joined-chatroom Event
This event's is actually designed for listening event message after somone has joined the Chat room (Group).
This event ("joined-chatroom") has to be listened to identify who joined the group. After listening to this event update the groups joined list and groups available list as well .
Example output it as follows:
                            {
                                chatRoomId:'unique chat room id',
                                chatRoomTitle: 'Name of chat room',
                                userId: 'userId of joiner',
                                userName: 'User who have joined this chat room', 
                            }
leaved-chatroom Event
This event's is actually designed for listening event message after somone has leaved the Chat room (Group).
This event ("leaved-chatroom") has to be listened to identify who leaved the group. After listening to this event update the groups joined list and groups available list as well .
Example output it as follows:
                            {
                                chatRoomId:'unique chat room id',
                                chatRoomTitle: 'Name of chat room',
                                userId: 'userId of user who have leaved the chat room ',
                                userName: 'User who have leaved this chat room', 
                            }
deleted-chatroom Event
This event's is actually designed for listening event message after somone has deleted the Chat room (Group).
This event ("deleted-chatroom") has to be listened to identify who deleted the group. After listening to this event update the groups joined list and groups available list as well .
Example output it as follows:
                            {

                                chatRoomId:'unique chat room id'
                                chatRoomTitle: 'Name of chat room',
                                userId: 'userId of user who have leaved the chat room ',
                                userName: 'User who have leaved this chat room', 
                            }
get-chat Event
This event's is actually designed for listening event message and getting message data after user has sent the message to chat room (Group).
This event ("get-chat") has to be listened to identify message details.
Example output it as follows:
                        {
                            senderName: 'Sender name who has sent a message',
                            senderId: 'Id of sender',
                            receiverName: 'Name of reciever', //for group chat it will be chatroom name
                            receiverId: 'Id of reciever',     //for group chat it will be chatroom id
                            message: 'Message that has sent',
                            chatRoom: 'Id of Chat Room',
                            chatRoomTitle:'Title of chat Room',
                            createdOn: 'Date and time of message sent ',
                            chatId : 'Unique id of chat'
                        }
tping Event
This event's is actually designed for listening who is currently typing the message in the group
This event ("typing") has to be listened to identify who is typing the message.
Example output it as follows:
                            {
                            	userName: 'Name of user',
                            }

Events to emit

set-user event
This event ("set-user") has to be emitted when a user comes online.
User can only be set as online only after verification of authentication token. Which you have pass here. The following data has to be emitted
authentication token
chat-msg event
This event ("chat-msg") has to be emitted to identify an individual chat message that has been send.
The following data has to be emitted
                            {
                                senderName: 'John Doe',
                                senderId: 'userId of sender',
                                receiverName: 'Smith',
                                receiverId: 'userId of receiver',
                                message: 'your chat message',
                                createdOn: date
                            }
mark-chat-as-seen event
This event ("mark-chat-as-seen") has to be emitted, when the user start chat with other user.
The following data has to be emitted
                            {
                                userId: 'userId of user',
                                senderId: 'userId of user from whom current user is chatting'
                            }
create-chat-room Event
This event ("create-chat-room") has to be emitted to create the chat room .
The following data has to be emitted
                            {
                                chatRoomTitle: 'Name of chat room',
                                chatRoomLink: 'link of chat room',
                                userId: 'userId of creator',
                                userName: 'User who have created this chat room', 
                            }
join-chat-room Event
This event ("join-chat-room") has to be emitted to join the chat room .
The following data has to be emitted
                            {
                                chatRoomId:'unique chat room id'
                                chatRoomTitle: 'Name of chat room',
                                userId: 'userId of creator',
                                userName: 'User who have created this chat room', 
                            }
leave-chat-room Event
This event ("leave-chat-room") has to be emitted to leave the chat room .
The following data has to be emitted
                            {
                                chatRoomId:'unique chat room id'
                                chatRoomTitle: 'Name of chat room',
                                userId: 'userId of creator',
                                userName: 'User who have created this chat room', 
                            }
delete-chat-room Event
This event ("delete-chat-room") has to be emitted to delete the chat room .
The following data has to be emitted
                            {
                                chatRoomId:'unique chat room id'
                                chatRoomTitle: 'Name of chat room',
                                userId: 'userId of creator',
                                userName: 'User who have created this chat room', 
                            }
share-chat-room Event
This event ("share-chat-room") has to be emitted to share the chat room link via email.
The following data has to be emitted
                            {
                            	emailId: 'Email id of User to whom you would like to send invite to join chat room',
                            	senderName: 'Name of sender',
                            	chatRoomTitle: 'Name of Chat Room',
                            	chatRoomLink: 'Link of Chat Room',
                            	chatCreatedBy: 'Name of Admin who have created the chat Room',
                            	baseUrlApplication:'Url of application ' //in terms of localhost it should be 'http://localhost:4200'
                            }
typing Event
This event ("typing") has to be emitted to identify who is typing the message.
The following data has to be emitted
                            {
                            	userName: 'Name of user',
                            }
